\section{Cursos}

\begin{itemize}
  \item \textbf{Nvidia CUDA: Programação Parallela em GPU} --- LNCC.

    Utilizando \textbf{C} como linguagem de programação nas primeiras
    versões do SDK.

  \item \textbf{Maratona Behind the Code} {[Jul-Ago, 2019 \& Jul-Ago, 2020]} --- IBM.

    Utilizando \textbf{IBM Cloud}, a inteligência artificial \textbf{Watson},
    e o framework blockchain \textbf{Hyperledger Fabric}, além de conhecimentos
    gerais sobre desenvolvimento de software, para solucionar desafios de negócios
    propostos por diferentes empresas do Brasil.

  \item \textbf{Java Programming Masterclass for Software Developers} --- Udemy.

    Utilizando OpenJDK \textbf{v1.8} e \textbf{v11}, cobrindo sintaxe de
    linguagem, API interna, padrões de configuração e código-fonte,
    gerenciamento de projeto com \textbf{Apache Maven}, plug-ins, arquétipos
    e dependências externas.

  \item \textbf{Machine Learning A-Z\texttrademark: Hands-On Python \& R In Data
    Science} --- Udemy.

    Utilizando Spyder e R Studio para criar modelos de aprendizagem de
    máquina a partir de conjuntos de dados públicos, com base em padrões
    da indústria, gerenciando versões de bibliotecas, principalmente
    \textbf{SciPy}, rodando em cima do \textbf{Jupyter Notebook}.

  \item \textbf{Docker and Kubernetes: The Complete Guide} --- Udemy.

    Utilizando técnicas de DevOps com aplicativos de diferentes campos
    da tecnologia, implementando integração e entrega contínua
    (\textbf{CI/CD}) em projetos, automação de pipelines, microsserviços
    de alta disponibilidade e contêineres minimalistas.

  \item \textbf{The Complete Angular Course: Beginner to Advanced} --- Udemy.

    Utilizando Angular \textbf{v2.0}+, agregando internacionalização,
    comunicação com API back-end, e criando aplicações web dinâmicas,
    implementadas em \textbf{TypeScript}.

\end{itemize}
